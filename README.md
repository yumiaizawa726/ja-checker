# 🇯🇵 日本語文章チェッカー

## 何をするツールか
日本語の文章を構造的に正しく評価するツール

## なぜこの設計にしたか
LLMを利用せずに確実に日本語の問題点を指摘するよう設計した。

- ルールベース（構文・主語）：GiNZAによる依存関係解析
- 意味ベース（embedding）：文間の論理的連続性を4指標で評価
- エラータイプ設計（論理分類）：弱い主張を6カテゴリに分類
- ユーザー行動ログ（採用/却下）：却下履歴から誤検出パターンを学習

## スコアの意味

| 指標 | 内容 | 満点 |
|---|---|---|
| 総合スコア | 主語明確性40% + 論理整合性60% - 各ペナルティ | 100 |
| 主語明確性 | 主語の有無と確信度の平均。不在で-20pt | 100 |
| 論理整合性 | 文間の論理飛躍。トピックジャンプで-30pt | 100 |
| 長文減点 | 80文字超の文1件につき-15pt（最大-30） | 0 |
| 構造減点 | 接続詞連続・複数動詞など（最大-30） | 0 |
| ポエム減点 | 抽象語・形容詞の過多（最大-30） | 0 |
